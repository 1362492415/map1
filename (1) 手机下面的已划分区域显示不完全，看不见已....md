### 1. 高德地图 API（地图显示与搜索服务）
对方需要去[高德开放平台](https://lbs.amap.com/)注册账号，创建一个应用，申请“Web端(JS API)”的服务。
**需要替换的文件：`index.html`**

*   **第 15 行**：把引入脚本链接中的 `key=4ca4a...` 替换为对方申请的高德 **Key**。

### 2. Bmob 后端云（数据存储服务）
对方需要去 [Bmob 官网](https://www.bmob.cn/) 注册账号，创建一个免费应用，用来存储他们自己画的区域数据。
**需要替换的文件：`js/store.js`**
*   **第 7 行**：找到 `Bmob.initialize("ba4204dffe52b0e7", "1234567890123456");` 这一行，把里面的两段字符替换为对方在 Bmob 后台拿到的 **Application ID** 和 **REST API Key**。
*   *注意：对方换上自己的密钥后，还需要在他们自己的 Bmob 后台设置里关闭“应用安全限制”（或者把他们最终生成的网页链接加到安全域名里），否则网页会无法保存数据。*

### 3. GitHub Pages（网页托管与网址生成）
你现在的网页是通过你的 GitHub 账号生成的 `1362492415.github.io` 链接。
*   **无需替换代码**，但对方需要**自己注册一个 GitHub 账号**。
*   对方拿到你发给他们的这整个文件夹后，需要像你刚才那样，把代码拖拽上传到他们自己的 GitHub 仓库中。
*   然后在他们自己的仓库设置（Settings -> Pages）里开启 GitHub Pages 服务，从而生成属于他们自己的访问链接。
